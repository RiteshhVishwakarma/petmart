## 📱 Installation Tips:

### Enable Unknown Sources:
**Android 8.0+:**
Settings > Apps > Special Access > Install Unknown Apps > Chrome/Files > Allow

**Android 7.0 and below:**
Settings > Security > Unknown Sources > Enable

### If Installation Blocked:
1. Check "Install from unknown sources"
2. Try different file manager
3. Transfer via USB and install
4. Use ADB install (advanced)
